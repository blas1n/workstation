# workstation

Scripts and configs for managing a multi-project development environment — worktree automation, port allocation, auto-deploy.

## Setup

```bash
git clone https://github.com/blas1n/workstation.git ~/Works/_infra

# Register launchd jobs
ln -sf ~/Works/_infra/launchd/com.blas1n.autodeploy.plist ~/Library/LaunchAgents/
launchctl load ~/Library/LaunchAgents/com.blas1n.autodeploy.plist
```

## Project Layout

All projects follow the bare repo + worktree pattern:

```
~/Works/<project>/
  .bare/          # bare git repo
  main/           # main worktree (deploy)
  wt/             # agent worktrees
    feature-foo/
    fix-bar/
```

## Port Allocation

Each project has a unique port range. Worktrees use a +10000 offset.

| Project | APP | FRONTEND | PG | REDIS | Other |
|---------|-----|----------|----|-------|-------|
| bloasis | 8000 | 3000 | 5432 | 6379 | - |
| BSGateway | 4000 | - | 5433 | 6380 | - |
| BSNexus | 8100 | 3100 | 5434 | 6381 | - |
| bsai | 8200 | 3200 | 5435 | 6382 | KEYCLOAK=8080 |
| BSForge | 8300 | - | 5436 | 6383 | - |
| BSage | 8400 | - | - | - | - |
| MetaSummarizer | - | - | - | - | CLI only |

Worktree port = `base + 10000 + (slot - 1)`:
- slot 1: bloasis APP=18000, FRONTEND=13000, PG=15432, REDIS=16379
- slot 2: bloasis APP=18001, FRONTEND=13001, PG=15433, REDIS=16380

## Scripts

### create-worktree.sh

Creates a git worktree with automatic port allocation.

```bash
# Auto-assign slot
scripts/create-worktree.sh bloasis feature/new-api

# Explicit slot
scripts/create-worktree.sh BSNexus fix/redis-bug 2
```

What it does:
1. Creates a worktree from `.bare/` (auto-creates branch if needed)
2. Auto-assigns slot 1–5 (tracked via `.wt-slot` marker file)
3. Generates `.devcontainer/.env` with `EXTERNAL_*_PORT` variables

Generated `.env` example:
```
# Auto-generated by create-worktree.sh (slot 1)
EXTERNAL_BACKEND_PORT=18000
EXTERNAL_FRONTEND_PORT=13000
EXTERNAL_PG_PORT=15432
EXTERNAL_REDIS_PORT=16379
```

### remove-worktree.sh

Removes a worktree and frees its slot.

```bash
# List worktrees
scripts/remove-worktree.sh bloasis

# Remove
scripts/remove-worktree.sh bloasis feature-new-api
```

Prompts to delete the branch after removal (skips main).

### status.sh

Shows all worktrees and running Docker containers across projects.

```bash
scripts/status.sh
```

### autodeploy.sh

Pulls and rebuilds when origin/main has new commits. Designed to run via launchd every 2 minutes.

```bash
scripts/autodeploy.sh
```

Targets: bloasis, BSGateway, BSNexus, bsai, BSForge, BSage (projects with `deploy/docker-compose.yml`).
Logs: `logs/autodeploy.log`
